Python API
==========

Module :py:mod:`spanlib`


Main classes of of :mod:`spanlib`:
    
    - :mod:`~spanlib.analyzer.Analyzer`: main class (PCA and MSSA).
    - :mod:`~spanlib.dual.Dual`: for dual analyzes (SVD).
    - :mod:`~spanlib.data.Data`: Management of a single variable.
    - :mod:`~spanlib.data.Dataset`: Management of a group of variables.


Import them with:
    
    >>> from spanlib import Analyzer
